What is CSS? An introduction to CSS

Last modified - 06-05-2023


 

What is CSS?


CSS (Cascading Style Sheets) is a way to make websites look pretty and fun. Just like how you might choose different colors and designs when you draw a picture, CSS allows website designers to choose different colors and designs for their websites.


Think of a website as a big coloring book. The HTML (Hypertext Markup Language) is like the outlines of the pictures, and CSS is like the colors and designs you use to fill in the pictures.


For example, if you wanted to make the text on a website bigger, you would use CSS to tell the website to make the text bigger. If you wanted to make the background of the website blue, you would use CSS to tell the website to use the color blue for the background.


CSS has lots of different things you can do with it. You can change the colors of things, make things bigger or smaller, move things around on the page, and much more!


It's kind of like a magic wand that lets you change the way a website looks, and you don't even have to be a wizard to use it!


I hope that helps you understand a little bit about what CSS is and what it does.


One of the most important things about CSS is that it allows you to make changes to a website without changing the underlying content. This means that you can make a website look different without having to change the words or pictures on the website.


For example, imagine you have a website with a picture of a cat. You could use CSS to make the picture bigger or smaller, change the color of the background, or add a border around the picture. None of these changes would affect the actual picture of the cat itself, but they would make the website look different.


CSS works by using something called selectors. Selectors are like little flags that tell the website which part of the page you want to change. For example, you could use a selector to tell the website to change the color of all the text on the page, or to change the font size of just one specific paragraph.


Once you've selected the part of the page you want to change, you can use CSS to apply different styles to that part of the page. Styles are like instructions that tell the website how to display the selected part of the page. You can use styles to change things like the color, font, size, and spacing of the text, as well as the background color or image of the page.


One of the great things about CSS is that you can use it to make a website look the way you want it to, even if you didn't create the website yourself. This means that you can use CSS to customize websites you visit, like your favorite game or social media sites, to make them look more fun and interesting.


How CSS works with HTML?


Let's say we have an HTML document that looks like this:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Welcome to my website!</h1>
    <p>This is some text on my website.</p>
    <img src="cat.jpg">            
</body>
</html>

This is a very simple HTML document with a heading, a paragraph, and an image. We're also linking to a CSS file called "style.css" using the link element in the head section of the HTML.


Now let's look at what the CSS code might look like in the "style.css" file:


body {
    background-color: #f0f0f0;
    }       
      
h1 {  
    color: blue;
    font-size: 32px;
    }       
      
p {  
    color: green;
    font-size: 24px;
    line-height: 1.5;
    }   
      
img {  
    border: 1px solid black;
    width: 50%;
    height: auto;
    }

Output



This CSS code tells the website to do the following things:


Set the background color of the entire page to a light greyish color (#f0f0f0).

Change the color and font size of the h1 (heading) element to blue and 32 pixels, respectively.

Change the color, font size, and line height of the p (paragraph) element to green, 24 pixels, and 1.5 times the font size, respectively.

Add a black border that's 1 pixel wide around the img (image) element, and set the width of the image to be 50% of its original size.


As you can see, CSS code is made up of selectors and styles. In this example, the selectors are body, h1, p, and img, and the styles are the properties that follow each selector (like background-color, color, font-size, and border). By combining selectors and styles in this way, we can make the website look the way we want it to!


Some basic CSS commands:


color: sets the color of text or other elements.

color: red;

background-color: sets the background color of an element.

background-color: yellow;

font-size: sets the size of the font.

font-size: 16px;

font-family: sets the font family (or typeface) for the text.

font-family: Arial, sans-serif;

text-align: sets the alignment of the text within an element.

text-align: center;

padding: adds space between the content of an element and its border.

padding: 10px;

margin: adds space outside of an element, between it and other elements.

margin: 20px;

border: adds a border around an element.

border: 1px solid black;

width and height: sets the width and height of an element.

width: 100px


height: 50px;

Font size: controls the size of the text displayed on a web page. 

font-size: 16px;

Conclusion


In summary, CSS is a way to make websites look pretty and fun. It works by using selectors to choose different parts of the page you want to change, and styles to apply different colors, fonts, sizes, and spacing to those parts of the page. CSS allows you to customize the websites you visit to make them look the way you want them to, and it's a powerful tool for website designers and developers.
