HTML for beginners, What is HTML?

Last modified - 18-01-2023

If you are working in website or software industry you may have heard the term HTML, so have you ever wondered what is HTML? Where it is used? In this article we are going to read about HTML, its attribute and some of its tags. The full form of HTML is Hypertext Markup Language. It is used to create and structure web pages. HTML uses a series of tags and attributes to define the structure and layout of a web page, it includes headings, paragraphs, images, and links. It is the backbone of any website.


What is HTML used for?


HTML is primarily used for creating the structure and layout of web pages. It provides a way to describe the content of a web page using a set of tags and elements, such as headings, paragraphs, images, and links. These elements are used to define the structure of the content on a web page, and can be styled using Cascading Style Sheets (CSS) to control the visual presentation and design of the page. HTML is used to create static web pages, which can be viewed in a web browser and do not change unless the source code is manually updated. HTML is also used as the foundation of many web-based applications and dynamic websites, which use scripting languages like JavaScript to create interactive and responsive user interfaces.



Basic structure of HTML code


The basic structure of an HTML code in known as HTML boilerplate, it includes the doctype declaration, head section (with meta information and links to CSS and JavaScript files), and body section. It may also include commonly used elements such as a header, navigation, and footer. Additionally, it may include some basic CSS styles and JavaScript code to help with things like responsive design and cross-browser compatibility. However, the exact contents of an HTML boilerplate can vary depending on the specific needs of the project and the preferences of the developer or the client.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
                    
</body>
</html>

Heading tag in HTML


In HTML, heading tags are used to create headings and subheadings in a document. There are six heading tags in html, with the most important headings being denoted by the <h1> tag and the least important by <h6> tag. Here is an example of how they are used:

<h1>This is a main heading</h1>
<h2>This is a subheading</h2>
<h3>This is a sub-subheading</h3>

The text within the heading tags is usually rendered in a larger and bolder font than the surrounding text, and it is used to organize the content of the document, it makes the content easy to understand. Search engines also use the heading tags to understand the structure and hierarchy of the content on a page, so it is important to use them correctly and in a meaningful way.


Paragraph tag in HTML


In HTML, the <p> tag is used to create a paragraph. The text that appears between the opening <p> tag and the closing </p> tag is rendered as a block of text with a default margin at the top and bottom. Here is an example of how it is used: 

<p>This is a paragraph. It can contain multiple sentences and it can be styled using CSS</p>

The <p> tag is a block-level element, which means it takes up the full width of its parent container for example a <p> tag is in written in the <body> section then it takes the full width of the section and creates a new line before and after the element. The text inside the <p> tag is usually rendered in the browser's default font and size, but it can be styled and changed using CSS.


Div tag in HTML


In HTML, the <div> tag is a block-level element like <p> tag, it brings other elements like heading paragraphs together it is useful for styling means you can style many elements using a single <div> tag. The <div> tag works like a container for other HTML elements. Here's an example of how the <div> tag is used:

<div>
    <p>This is some text in a paragraph element</p>
    <p>This is some text in an another paragraph element</p>
<div>

In this example, the <div> tag is used to group two <p> tags together. This allows you to apply styles to both paragraphs at once, instead of applying the styles to each paragraph separately.

The <div> tag does not have any default formatting like margin and padding, so it's often used as a container for other elements. You can add styles to the div element by assigning it a class or id and target it with CSS.


It's important to note that when using <div> tags to group elements, semantic tags like <header>, <nav>, <main>, <article>, <section>, <aside>, <footer> should also be used when required. These elements not only help with styling but also give the web page a meaningful structure.


Img tag in HTML


In HTML, the <img> tag is used to embed images in a web page. The <img> tag is a self-closing, empty tag, which means it doesn't have a closing tag and doesn't contain any content. Here's an example of how the <img> tag is used:

<img>img src="image.jpg" alt="A description of the image"</img>

The src attribute is used to specify the URL of the image to be displayed. The alt attribute is used to provide a text description of the image it helps in the seo of image and the web page, it will be displayed if the image is not loaded. The <img> tag also has other attributes such as width, height, style, class, etc. You can use these attributes to control the size and appearance of the image.


It's important to note that the <img> tag is not a semantic element, it's used to embed images in the web page, it doesn't give any meaning to the content. If you want to include an image with a caption or a figure with something, you can use the <figure> and <figcaption> elements.
